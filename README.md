# MyTrello Backend

### How to run server?

```shell
make api-up
```

### How to stop server?

```shell
make stop
```

### How to clear all absolutely (delete all containers)?

```shell
make down
```

## Technologies:

- Golang
- Postgres, Replication Primary-StandBy
- Redis
- S3 storage
- Nginx
- Docker, Docker Compose
- Allure
- Prometheus, Node Exporter
- Grafana
- OpenTelemetry
- Jaeger
- GitHub Actions (CI)
- OpenAPI, Swagger
